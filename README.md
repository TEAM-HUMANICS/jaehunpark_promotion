# 박재훈 × SEGYM 프로모션 페이지

휴머닉스 세짐과 보디빌더 박재훈(올라잇)의 콜라보레이션 프로모션 랜딩 페이지입니다.
QR코드 및 인스타그램 링크 유입 고객을 **구매 상담 전환**으로 이끄는 것이 목표입니다.

## 디자인 컨셉: Hardcore High-Tech

- **Void** `#0A0A0A` - 심연의 블랙 배경
- **Ember** `#E63946` - 불타오르는 레드 (CTA, 강조)
- **Steel** `#3B82F6` - 정밀한 메탈릭 블루 (데이터, 기술)

## 기술 스택

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React (아이콘)

## 실행 방법

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 주요 기능

1. **Hero Section** - 로봇 설계도 라인 아트 오버레이
2. **Data-Driven Proof** - 스크롤 반응 애니메이션 차트
3. **All-Right Routine** - 박재훈 선수 실제 세짐 설정값
4. **Fire Particles / Smoke** - 불꽃 파티클 & 연기 효과
5. **Sticky CTA** - 하단 고정 "박재훈 전용 혜택 받기" 버튼
6. **FOMO Toast** - 실시간 상담 신청 알림 (예: "방금 올라잇짐에서 혜택 상담을 신청했습니다")
7. **Scroll Snap** - 모바일 섹션별 스냅 스크롤
